# 🚀 CopyClip v0.4.0

**Project Intelligence Dashboard + Intent Authority MCP Server**

CopyClip is the bridge between human architectural intent and AI-agent execution. In an era where most code is generated by agents, CopyClip ensures the human developer maintains 100% cognitive connection with the project.

---

## 🌟 Core Pillars

### 🧠 Project Consciousness (GenUI)
Forget static dashboards. Interact with your project through a **Generative UI Chat**. 
- Ask about architecture, risks, or logic.
- The agent injects **Dynamic Artifacts**: 3D Maps, Risk Heatmaps, and Intent Manifestos directly into the conversation.

### 🌐 The Atlas (Cosmic 3D Engine)
Visualize your codebase as a living universe powered by **Three.js**.
- **Sitting Tree Algorithm:** Folder hierarchy translated into galactic rings and vertical layers.
- **Fog of War:** Real-time visualization of "dark code" (generated by agents, unreviewed by humans).
- **Planetary Metrics:** Nodes scale and glow based on cognitive debt and complexity.

### 📜 Intent Authority (MCP Server)
CopyClip acts as the **Central Intent Oracle** for external agents (Claude Code, Gemini CLI, Cursor).
- **Semantic Audit:** Agents must validate their proposals against your "Intent Manifesto" before execution.
- **Bidirectional Proposals:** Agents can propose new architectural decisions directly to your Planning Kanban.
- **Context Compression:** High-signal, token-optimized context bundles anchored to human decisions.

### ⏳ The Chronicle (Event Timeline)
A unified feed merging Git history, architectural decisions, and narrative snapshots. Understand the **Why** behind the **What**.

### 📋 Planning Kanban
Manage the future of your architecture. Move AI proposals from "Intentions" to "Anchored" status as they become part of the project's soul.

## 🛠 Quick Start

### 1. Installation

**macOS / Linux** (one-liner):
```bash
curl -fsSL https://raw.githubusercontent.com/sssamuelll/copyclip/main/install.sh | bash
```

**Windows** (PowerShell):
```powershell
irm https://raw.githubusercontent.com/sssamuelll/copyclip/main/install.ps1 | iex
```

**With pipx** (recommended for Python developers):
```bash
pipx install "copyclip @ git+https://github.com/sssamuelll/copyclip.git"
```

**With pip** (manual):
```bash
pip install "copyclip @ git+https://github.com/sssamuelll/copyclip.git"
```

**From source** (development):
```bash
git clone https://github.com/sssamuelll/copyclip.git
cd copyclip
python3 -m pip install -e '.[dev]'
```

### Updating

From the CLI:
```bash
copyclip update
```

Or re-run the install script — it detects the existing installation and upgrades automatically.

### 2. Start the Consciousness
```bash
copyclip start
```
*Note: If it's your first time, CopyClip will guide you through an interactive LLM setup (DeepSeek, OpenAI, Anthropic, etc.) and perform an initial project analysis.*

### Local development (green path)
From source, the canonical local development path is:

```bash
python3 -m pip install -e '.[dev]'
npm --prefix frontend install
./scripts/dev-smoke.sh
copyclip start --no-open --path .
```

See `docs/LOCAL_DEVELOPMENT.md` for the full verified local setup flow and the current smoke-check entrypoint.

### 3. Connect External Agents (MCP)
Add CopyClip to your `mcp_servers` configuration:
```json
"copyclip": {
  "command": "copyclip",
  "args": ["mcp"]
}
```

You can verify the command locally with:
```bash
copyclip mcp --help
```
