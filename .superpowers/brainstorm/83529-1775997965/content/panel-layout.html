MY RECOMMENDATION
C — Toggleable Split. The augmented-ui panel already appears on select. Instead of adding a second panel, we expand it: info section on top, code section below, all within the same augmented-ui frame. On hover it stays compact (just info). On click/select it expands to show code. This keeps the Atlas view clean (no extra panels eating space) and maintains one cohesive augmented-ui element.
